# Pollify

Pollify is a web app that allows you to create, share, and monitor the results of polls â€”> including custom and image-based polls.

I built Pollify because I wanted more flexibility than existing polling platforms offer. Most tools felt limiting, especially for visual or non-standard polls, so I created something that supports more customizable polling use cases.

---

## Features
- Create custom polls (including image-based polls)
- Share polls via link
- Monitor poll results in real time
- Simple, clean UI

---

## Tech Stack
- Next.js
- Supabase (database + authentication)
- Vercel (deployment)
